## Live Demo

> This app is available through [this link.](https://evolution-simulation-v2.netlify.app/)

## Running

In order to run this project locally, clone the repo and:

1. Install the NPM dependencies:
```
    npm install
```
2. Run the development server:
```
    npm run dev
```
